# ⚡ SCCM Logo Converter

A lightweight, browser-based tool for converting logos into **256x256** and **512x512 PNGs**, ready for **SCCM application packaging**.

No installs. No PowerShell. Just drag, click, and download.

---

## 🚀 Live Demo  
👉 <a href="https://davidjenner.github.io/SCCM-LogoConverter/" target="_blank">Launch the Logo Converter</a>

---

## 🧠 Why This Exists

When packaging applications in SCCM, getting logos into the correct format and size can be unnecessarily time-consuming.

This tool removes that friction by:
- Automatically resizing images  
- Maintaining aspect ratio  
- Exporting clean PNGs in the required dimensions  

---

## ✨ Features

- 📂 **Drag & Drop Upload**  
- 🖼️ **Instant Image Preview**  
- 📏 **True 256x256 & 512x512 Outputs**  
- 🎯 **Aspect Ratio Preserved (No Distortion)**  
- ⚪ **White Background Fill for Transparency**  
- ⬇️ **One-Click Downloads for Each Size**  
- 🌐 **Fully Browser-Based (GitHub Pages)**  

---

## 🛠️ How It Works

1. Upload or drag your logo into the tool  
2. Preview the image instantly  
3. Click:
   - **Download 256x256**
   - **Download 512x512**
4. Your PNGs are generated and downloaded instantly  

All processing happens locally in your browser using the **HTML5 Canvas API** — no files are uploaded anywhere.
